# 🏠 SmartBridge Housing Dashboard (Tableau)

This repository contains a Tableau dashboard analyzing housing data, developed as part of a data visualization project.

## 📁 Files

- `project-smartbridge.twb`: Tableau workbook containing all visualizations.
- `Transformed_Housing_Data2.csv`: Transformed and cleaned housing dataset used in the project.

## 📊 Dashboard Overview

The dashboard provides insights into:
- Housing price trends
- Property types distribution
- Location-based price analysis
- Other key performance indicators (KPIs)
