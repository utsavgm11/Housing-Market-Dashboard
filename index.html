Project Overview

Our project focuses on interactive dashboards and visual insights built using Tableau, paired with a fully responsive and user-friendly website developed using HTML, CSS, JavaScript, and Bootstrap.

Utsav Mahyawanshi played a leading role in the project by handling complete website development, UI/UX integration, and front-end deployment. Additionally, he was actively involved in data preprocessing, dashboard structuring, and ensuring seamless integration between the website and Tableau visualizations.
